# Web1 Leyu Hub

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为静态页面的集合与展示空间。

## 项目简介

本仓库主要用于存放各类独立的 HTML 页面，这些页面可能涵盖工具、演示、学习笔记或实验性内容。每个页面均以单一 HTML 文件形式存在，便于直接查看或部署。

## 目录说明

```
.
├── pages/          # 存放独立 HTML 页面文件的目录
├── assets/         # 页面可能引用的资源文件（如 CSS、JS、图片等）
├── index.html      # 主入口页面（可选）
└── README.md       # 本说明文件
```

- `pages/` 目录下每个子目录或文件代表一个独立页面，命名清晰，便于识别。
- `assets/` 目录用于存放页面所需的公共资源，按需组织。
- 根目录下的 `index.html` 可作为导航或欢迎页面。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部后端服务。
- 页面之间无强关联，可单独访问或部署。
- 内容以静态展示为主，部分页面可能包含简单的交互功能。
- 页面代码遵循基本 Web 标准，尽量保证在主流浏览器中正常显示。

## 维护说明

- 本仓库不定期更新，新增页面或修正已有内容。
- 欢迎通过 Issue 提出建议或报告问题，但请注意本仓库不针对具体网站或服务。
- Pull Request 接受合理改进，但请保持页面内容与仓库定位一致（即独立、通用的 HTML 页面）。
- 由于页面均为静态文件，部署方式灵活，可通过 GitHub Pages 或其他静态托管服务发布。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件。
3. 也可将整个仓库部署至静态服务器，通过 URL 访问各页面。
